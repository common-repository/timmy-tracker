== Description ==

Do you win everytime you roll up the rim? 
Let the readers of your blog know how well you are doing. Track your Tim Horton's Roll Up The Rim To Win purchases, wins and losses with the Timmy Tracker widget.

Features

*   Track purchases, wins and losses
*   Track small, medium, large and extra large cup size
*   Show or hide statistics table

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the Timmy Tracker widget over to your sidebar.
